What is this? Kobo renders sideloaded EPUBs through Adobe’s legacy “RMSDK” engine (CSS frozen ~2013). One modern value token — calc(), min(), clamp(), var() — makes it drop your entire stylesheet or refuse to open the book, even though the CSS is valid EPUB3. kobofix removes/resolves those tokens, converts rem/viewport units, fixes mimetype packaging, and flags layout features (flexbox/grid) that need a human. Inspired by this write-up.

Rule ids: KOBO-000 packaging · KOBO-001 value functions · KOBO-002 viewport units · KOBO-003 empty @media · KOBO-004 rem · KOBO-010+ flexbox/grid/transform (manual review). The same checks run as a CLI and merge with EPUBCheck — see the project README.
